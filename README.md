## Technologies Used

### Frontend
- **Next.js** (App Router)
- **React** (v18)
- **TypeScript**
- **Tailwind CSS** (with custom theme)
- **shadcn/ui** component library
- **Framer Motion** for animations
- **React Hook Form** for forms
- **Zod** for validation

### Backend
- **Nextjs**
- **MongoDB** (with Mongoose)
- **JWT** authentication
- **Redis** for caching
- **AWS S3** for storage
